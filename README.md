# ros-robosys2018

# Overview
PCからweb上にデータ（数値）を送り、表示する。

# Description
rosbridgeを使いパブリッシャ(count_py)からパブリッシュされた数値をweb上に送り、表示する。

# Demo


# Requirement
* Computer
  * PC
* OS
  * Ubuntu16.04.5(VirtualBox上で可)


# Install
```
$ git clone https://github.com/Yu0097/ros-robosys2018.git
```
